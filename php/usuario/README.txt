- conexion_bi.php (tu conexión actual)

## Importante (tabla)
Tu tabla original no tiene autoincremento en id_usu. Recomendado:

ALTER TABLE public.usuario
  ALTER COLUMN id_usu ADD GENERATED BY DEFAULT AS IDENTITY;

Si NO lo hacés, el sistema calcula id_usu = max+1 como fallback.

## Seguridad
